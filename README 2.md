Направляю ТЗ по проекту:

Идея: создать удобный веб-сервис для поиска и аренды жилья, где пользователь сразу видит все доступные варианты на карте и в списке, может быстро отфильтровать по нужным параметрам и связаться с собственником.

Как это должно выглядеть:

Главная страница  
Наверху:

- Логотип
- Кнопки «Войти» и «Регистрация»
- Иконка «Избранное»
- Навигация по типам жилья (Комната / Квартира / Дом)

Основная часть:

- Большая интерактивная карта с метками объектов (на каждой метке — цена)
- Список объявлений под картой или рядом: фото, краткое описание, цена, район/метро, кнопка «Подробнее» и «В избранное»
- Фильтры слева или сверху:
  - Тип жилья
  - Цена (от и до)
  - Район или станция метро
  - Количество комнат
  - Питомцы, дети, мебель и т.д.

Без авторизации можно просматривать объявления, но добавлять в избранное и писать собственнику — только после входа.

Карточка (страница) самого жилья:
- Фотографии
- Описание
 ⁃ Цена за 1 ночь 
 ⁃ Календарь выбора дат аренды
 ⁃ Адрес и возможность посмотреть на карте
 ⁃ Кнопка «Связаться с арендодателем» - личные сообщения через диалоговое окно сервиса
 ⁃ Кнопка «Забронировать»

Личный кабинет пользователя:

- Избранные объекты
- Настройки поиска (бюджет, предпочтения)

Админ-панель (для собственников и сотрудников агентства):

- Список всех своих объявлений
- Добавление нового:
  - Тип жилья
  - Цена
  - Загрузка фотографий
  - Описание
  - Указание адреса (точка на карте)
- Редактирование и удаление существующих объявлений

Регистрация и вход для пользователя — по email/телефону и паролю.
Регистрация и вход для арендодателя — по email/телефону и паролю. В ЛК арендодателя предусмотреть на будущее поле загрузки ID документа (российский паспорт) для отправки нам для верификации

Дизайн — чистый, светлый, адаптивный под мобильные устройства.

После запуска базовой версии можно добавить ИИ-функции:

- умный чат-помощник для подбора вариантов
- автоматический перевод объявлений (англ/рус)
- генерация описаний по фото
- анализ тональности отзывов
